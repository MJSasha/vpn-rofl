# VPN ROFL Manager

Простой веб-интерфейс для управления правилами и прокси-группами в конфигурации **Mihomo (Clash)** через SFTP.

## Особенности
- Редактирование правил `rules` (включение/выключение, изменение типа и группы).
- Управление прокси-группами `proxy-groups`.
- Автоматическая перезагрузка конфигурации Mihomo через API.
- Встроенная система самообновления.

## Требования
Для работы приложения необходим доступ к серверу по **SFTP** (SSH), где установлен Mihomo.

## Установка и запуск

### 1. Клонирование репозитория
**Важно:** Для работы функции обновления приложение должно быть склонировано через `git`. Не скачивайте архив, используйте команду:

```bash
git clone <ссылка_на_репозиторий> vpn-rofl
cd vpn-rofl
```

### 2. Настройка переменных окружения
Создайте файл `.env` на основе `.env.example` и заполните данные для доступа к вашему SFTP-серверу:

```env
SSH_HOST=127.0.0.1
SSH_PORT=22
SSH_USER=root
SSH_PASS=your_password
CONFIG_PATH=/etc/mihomo/config.yaml
MIHOMO_API=http://127.0.0.1:9090
```

- `SSH_HOST`, `SSH_PORT`, `SSH_USER`, `SSH_PASS` — данные для подключения к серверу, где лежит конфиг.
- `CONFIG_PATH` — абсолютный путь к `config.yaml` на удаленном сервере.
- `MIHOMO_API` — адрес внешнего контроллера Mihomo для применения изменений.
